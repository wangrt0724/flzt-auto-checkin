# FLZT

机场地址：

- [https://flzt.top](https://flzt.top)
- [https://fljc.cc](https://fljc.cc)

## 登录

请求网址: `
https://flzt.top/api/v1/passport/auth/login`

请求方法: `POST`

表单数据: `email=xxx&password=xxx`

响应

```json
{
  "status": "success",
  "message": "操作成功",
  "data": {
    "token": "",
    "auth_data": "Bearer xxx",
    "is_admin": false
  },
  "error": null
}
```
